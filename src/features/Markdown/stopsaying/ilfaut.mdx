**Il est de première importance d'** arrêter de répéter toujours **"il faut"**!
Pour mieux parler français, pour être écouté et peut être pour influencer, il faut... non! **il est primordial d'** apprendre des nouvelles expressions et formules. Alors voici encore quelques alternatvies pour exprimer _la nécessité ou l'obligation_:

- **_"Il est importantissime"_** : ce sufixe _"issime"_ est superbe, tu peux l'utiliser avec des **adjectifs**. Ça s'appelle _"le superlatif"_ → richissime, rarissime...

* **_"Il est impératif"_**

* **_"Il est super important de_** : les _francais_ aiment beaucoup utiliser ses mots: **super, hyper, mega** avant un adjectif ou un adverbe.

* **_"Il est nécesaire de"_** réviser un peu le subjonctif car il est utile ici dans les expressions que nous avons vues. _En effet, on doit utiliser le subjonctif après_ **_que_** _quand on exprime:_
  - une nécessité/une obligation: **_il est de première importance_** que tu _apprennes_.
  - un souhait: **J'aimerais** que vous _compreniez_ l'utilisation du subjonctif.
  - un ordre: **J'exige** que vous _parliez_ français avec moi!
  - une émotion: **Je suis heureux** que vous _appreniez_ le fançcais.
  - une opition à la forme négative: **je ne pense pas** que le français _soit_ une langue difficile.
  - le doute: **je doute** qu'il _abondonne_ un jour le français.
  - une possibilité: **Il se peut** que tu _parles_ couramment le français dans quelques mois.

Bravo pour ta progression en français.
