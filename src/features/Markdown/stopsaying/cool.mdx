NE DIS PAS **_"COOL"_**. C'est le signe d'un vocabulaire médiocre! Dis plutôt:

- **_"Mirifique"_** : "La vue depuis le sommet de la montagne était _cool_ ? **FAUX** - Elle était _mirifique_"

- **_"Grandiose"_** : "Le coucher de soleil sur la plage était _grandiose_."

- **_"Éminent"_** : "Son sens du style est _éminent_."

- **_"Stupéfiant"_** : "Cette performance était tout simplement _stupéfiante_."

- **_"Éblouissant"_** : "La manière dont vous avez orchestré cette soirée était _éblouissante_."

- **_"Épique"_** : "C'était une compétion _épique_ pour la première place."

- **_"Remarquable_** : "Cette solution que vous avez proposée est _remarquable_."

- **_"Spectaculaire"_** : "Le feu d'artifice du Nouvel An à paris était _spectaculaire_ !"

- **_"Prodigieux"_** : "Il a accompli un travail _prodigieux_ en un temps record !"

- **_"Admirable"_** : "Vous avez montré un effort _admirable_ en surmontant ces difficultés."

- **_"Sensationnel"_** : "Le dernier film qu'il a réalisé était _sensationnel_."

- **_"Fascinant"_** : "Votre idée est _fascinante_ !"

- **_"Superbe"_** : "Elle portait une robe _superbe_ !"

- **_"Phénoménal"_** : "Son progrès dans l'apprentissage du piano est _phénoménal_ !"

- **_"Époustouflant"_** : "La fin du concert était époustouflante !"

Oubliez donc ce sempiternel **_"cool"_**.
